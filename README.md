## Project SUHAY: Volunteer, Donations, and Inventory Management System

A work-in-progress system analysis and design school project developed to streamline and centralize the management of volunteers, donations, and inventory for organizations. The system is built with multiple user roles to support structured workflows and improve data organization and tracking.

This project is developed using the Laravel framework and focuses on designing a scalable and efficient information system based on real-world NGO operational needs gathered through stakeholder analysis and interviews.

## Tech Stack

- **Framework:** Laravel  
- **Database:** PostgreSQL
